# MyPagingThree

* [Room Database](https://developer.android.com/jetpack/androidx/releases/room?gclid=CjwKCAiAv9ucBhBXEiwA6N8nYLtgnha5YTMWwUQethxmwhOTCXgOVtd7JOpj75LL4vmSUiTcMM_gbxoC8XQQAvD_BwE&gclsrc=aw.ds)
* [BuildSrc Kotlin DSL](https://medium.com/android-dev-hacks/kotlin-dsl-gradle-scripts-in-android-made-easy-b8e2991e2ba)
* [Modern Storage Library](https://google.github.io/modernstorage/)
* [ktLint](https://pinterest.github.io/ktlint/)
